# Add Component

Create a new shadcn/ui-based component for the GIFTSSITE project.

## Instructions

1. First, use Context7 MCP to get latest shadcn/ui documentation
2. Check if the component already exists in `src/components/`
3. Create the component following these rules:
   - Use TypeScript with proper interfaces
   - Use Tailwind CSS classes only (dark theme)
   - Use shadcn/ui primitives where applicable
   - Server Component by default, "use client" only if interactive
   - Export from index file if exists
4. Add the component to the relevant page/layout
5. Run the ui-designer agent for design review

## Arguments
$ARGUMENTS = component name and description
